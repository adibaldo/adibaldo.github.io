# 📜 CADERNO DE EXPERIÊNCIA DO OSCAR (UI/UX)

## Estrela Guia: gwern.net
A partir de 16/02/2026, nossa meta é atingir o nível de excelência de leitura do site do Gwern.

### Princípios de Ouro:
1. **Tipografia é tudo:** O texto deve respirar. A fonte deve ser escolhida para longas leituras (Serif para o corpo, preferencialmente).
2. **Foco no Conteúdo:** Remova qualquer distração. O blog do seu Adi é um livro aberto, não um outdoor.
3. **Funcionalidade Silenciosa:** Sidenotes e pop-ups de links devem ajudar quem quer se aprofundar, sem atrapalhar quem quer só seguir a prosa.
4. **Resistência ao Tempo:** O código deve ser limpo e o design deve parecer atual daqui a 20 anos.
