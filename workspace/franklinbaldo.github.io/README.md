# The Chronicle of Franklin Baldo

This is the source code for [franklinbaldo.github.io](https://franklinbaldo.github.io), Franklin Baldo's personal blog and digital garden.

## Tech Stack

- **Framework**: [Astro 5](https://astro.build/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Typography**: EB Garamond (serif) + JetBrains Mono (monospace)
- **Deployment**: GitHub Pages via GitHub Actions

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Structure

- `src/content/blog/`: Markdown files for blog posts
- `src/pages/`: Astro pages and routes
- `src/layouts/`: Layout components
- `src/styles/`: Global styles and Tailwind configuration
