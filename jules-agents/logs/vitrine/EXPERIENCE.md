# Vitrine - Zelador Técnico do Blog

## Registro de Experiência e Aprendizado

### Descobertas sobre Estrutura do Astro/Markdown
- **Hero Images:** O campo `heroImage` é definido como uma imagem local, e o campo `heroImageAlt` é opcional, mas recomendado para acessibilidade.
- **Astro Build:** O processo de build verifica se as imagens referenciadas existem no sistema de arquivos.
- **Content Collections:** O esquema do blog é definido em `src/content.config.ts`.
